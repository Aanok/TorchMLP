gnuplot_cup (traces, nn)
Generate EPS plots for ML-CUP. The file name will tell about the employed hyperparameters.

Parameters

  • traces: Standard error/accuracy trace table. traces.validation is optional.
  • nn: Neural network that generated the traces. Hyperparameters are read from it.
gnuplot_monks (traces, nn, num, is_test)
Generate EPS plots for a MONKS problem. The file name will tell about the employed hyperparameters.

Parameters

  • traces: Standard error/accuracy trace table. traces.validation is required.
  • nn: Neural network that generated the traces. Hyperparameters are read from it.
  • num: MONKS dataset enumerator (1, 2 or 3)
  • is_test: Boolean indicating if traces.validation represents a Test set or Validation set.
parse_cup (source_file, has_labels)
Parse ML-CUP data file Parses provided CSV file, applying normalization to input and targets.

Parameters

  • source_file: Path to file with ML-CUP data to parse.
  • has_labels: Boolean to distinguish labelled (TR) and unlabelled (TS) sources.

Return value:

input,targets, t_min, t_max where
  • input: Array-like table with Tensors of input patterns.
  • targets: Array-like table with Tensors of target patterns. nil if has_labels == false.
  • t_min: Minimum of target patterns. nil if has_labels == false.
  • t_max: Maximum of target patterns. nil if has_labels == false.
